# 20250423 ch07 - Loop06 ~ ch09

## ch07
_주요 내용 : 별찍기_

순서대로 별찍기
``` java
for (int i = 0; i < 5; i++) {
    for (int j = 0; j < i; j++) {
        System.out.print("*" + " ");              
    }
    // 개행이 일어나는 1차 for문
    System.out.println();                   

```
결과   
i < 5이기 때문에 5줄로 별이 찍힌다.
``` java
    *
    * *
    * * *
    * * * *
    * * * * *
```
위와 반대로 별 찍기
